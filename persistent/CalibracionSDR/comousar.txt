Se cuenta con tres archivos .grc en esta carpeta, así como un archivo de Python.
1. "entender_ishort2complex.grc" es un flowgraph de GNU Radio realizado con
el objetivo de entender cómo funciona el bloque ishort2complex.
2. "transmisionTono.grc" es un flowgraph de GNU Radio que permite transmitir
un tono de amplitud variable con un HackRFOne.
3. "pluto_rx.grc" es un flowgraph de GNU Radio que imprime la potencia de las muestras
que va recibiendo, y sirve como referencia para ver que potumbral esté funcionando de manera
correcta.
4. "igualarpotencias.py" contiene el código de Python a utilizar en el Python block del
flowgraph del punto 3.
